Проект реализует простой REST-API для работы со списком задач (TODO-лист). Написан на языке Go, использует базовую структуру с хранилищем (SQLite или файл) и маршрутизацией HTTP-запросов.

## Возможности  
- Создание, чтение, обновление и удаление задач (CRUD).  
- Хранение данных локально (файл `data.db`) или иным хранилищем (можно доработать).  
- Минимальный и понятный код — удобен для обучения и расширения.
